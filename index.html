Questions We Always Get Asked

How long does Braille conversion take?
+
Instant ‚Äî if content is already digital, the ESP32 converts text to Braille commands in real-time. Zero delay between a student requesting a book and reading it in Braille.
What if the device breaks?
+
Every component is available from local Indian electronics vendors. Any IT student or basic technician can repair it. We've specifically designed for repairability, not planned obsolescence.
Can sighted students use this too?
+
Absolutely! We're building screen reader and audio output options so this device serves all students ‚Äî including those with partial vision, dyslexia, or other learning differences.
How does the school WiFi sync work?
+
The device connects to the school's existing WiFi network. Teachers upload lessons to a central server, and all student devices download automatically. No cables, no manual transfers, no delays.
Is this project open source?
+
Yes! Schools, students, and developers can access our hardware schematics and software. We want this to be a community effort that spreads organically across India.
